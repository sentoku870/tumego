# tumego（Web版）目的とスコープ

> 最終更新: 2025-12-30
> このドキュメントは **変更しにくい「固定ルール」** を定義します。
> バージョン計画は `30-roadmap.md` を参照してください。

---

## 1. 目的（このアプリで達成したいこと）

**Webブラウザ上で動く「詰碁・局面の編集＋解答＋共有」に特化した軽量ツール。**

具体的には：
- 本や野狐のスクリーンショットから、サッと局面を起こして検討する
- 問題図として固定して、自分や他人に解いてもらう
- 画像 / URL / QR で共有する

目的は「汎用SGFエディタ」ではなく、**「詰碁＋共有」に特化したWebツール**。

---

## 2. 想定ユーザー

- **主に**: 本やネットの詰碁をよく眺める級位者〜有段者
- **特徴**:
  - KaTrain / Lizzie / Sabaki は知っているが、起動が重い/入力が面倒
  - 「ちょっとした詰碁・局面」を扱うにはオーバーキルだと感じている
  - Discord 等で野狐の局面にコメント・解答を返すことがある

---

## 3. 想定ユースケース

### 3.1 本・スクショから局面をサッと再現（編集モード）
- 本の詰碁図・野狐のスクショを素早く再現
- その場で手を並べて読む（SGF保存不要なケースも多い）

### 3.2 問題図を作って解かせる（解答モード）
- 編集モードで作った局面を「問題図」として固定
- 共有方法：
  - A: 番号付き画像
  - B: URL / QRコード
  - C: 解答手順付きURL/QR

### 3.3 野狐スクショ → 解答図 → Discord共有
- 質問者は「野狐のスクショ画像」を投げるだけ
- 解答者はこのアプリで盤面を再現、解答を返す

### 3.4 外部SGFの軽いハブとして
- 外部SGFを読み込み → 局面再生 → 特定局面を問題図化

---

## 4. コア原則

### 4.1 シンプル優先
- ボタン・機能は「本当に使うもの」に絞る
- 迷いやすいモードや複雑な機能は、積極的に削る/設定の奥にしまう

### 4.2 役割分担
- **本格検討・AI解析**: KaTrain / Lizzie / Sabaki に任せる
- **このアプリ**: 「軽い編集・解答・共有」「スクショ→問題図」に集中

### 4.3 SGFの慎重な扱い
- originalSGF は読み取り専用、上書きしない
- 問題図・解答用SGFは別枠（problemSGF / solutionSGF）で扱う

---

## 5. スコープ（やること）

### 5.1 編集モード
- 石の連続配置（黒白切替がスムーズ）
- 最終的には囲碁ルールONで局面を整える

### 5.2 解答モード
- 問題図の固定と解答シーケンスの記録
- 番号付き画像への出力
- URL / QRコードでの共有

### 5.3 最低限のSGF機能
- 外部SGFの読み込みと単純な再生
- タイトル・対局者名・日付の軽い編集
- SGFのコピー（クリップボードへ）
- 石番号や簡単なマーク（△や○程度）

---

## 6. non-goals（やらないこと）

| やらないこと | 理由 |
|-------------|------|
| 大規模な棋譜管理 | フォルダ/タグ/検索は本格ソフトに任せる |
| 複雑な変化図ツリー編集 | 多数の分岐・注釈の階層は対象外 |
| 詳細なSGFプロパティ編集 | 棋戦名・タイマー設定等のフル編集は不要 |
| AIエンジン連携・勝率グラフ | 解析はPC版の役割 |
| 印刷用レイアウト | 紙媒体向け機能は対象外 |
| 多数のキーボードショートカット | シンプル優先 |

## 7. 設定（Preferences）の方針

### 7.1 設定に集約するもの
人によって好みが分かれるものは設定に集約：

| 項目 | 設定キー | デフォルト |
|------|----------|-----------|
| 囲碁ルール | `rulesMode` | `standard` |
| 抜いた石の表示 | `showCapturedStones` | `on` |
| 解答モードの全消去 | `allowFullResetInSolve` | `off` |

### 7.2 保存方法
- `localStorage` でブラウザローカルに保存
- 「設定をリセット」でデフォルトに戻せる

## 8. 野狐スクショ読み取り機能の段階

### 8.1 軽量版（ブラウザ内）
- 対象: 野狐の局面スクショ
- 目的: 盤面を手入力する負担を減らす
- 精度: 8〜9割読めればOK → 編集モードで微修正
- 要件: Web ブラウザ内で完結（JS/WASM）

### 8.2 PC専用の高度版（将来・拡張）
- 位置づけ: PC版の拡張機能
- 対象: 高性能ローカルPC
- このアプリのスコープ外
